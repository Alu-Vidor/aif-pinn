# AIF-PINN

Репозиторий с минимальной реализацией Asymptotically-Informed PINN (AIF-PINN) для решения фракционного варианта модели Солоу с памятью. Код разбит на модули, поэтому легко расширять компоненту генерации данных, операторы дробных производных, нейросеть и процедуру обучения.

## Возможности
- генерация коллокационной сетки с адаптивным разрешением в пограничном слое;
- оператор дробной производной Капуто по схеме L1;
- PINN c асимптотическими признаками и жёстким выполнением начального условия;
- двухэтапное обучение (Adam -> L-BFGS) и визуализация решения вместе с референсом Миттага-Леффлера.

## Установка
1. Установите Python 3.10+.
2. Установите зависимости (минимально необходимы `torch`, `numpy`, `matplotlib`):
   ```bash
   pip install torch numpy matplotlib
   ```

## Запуск эксперимента
```bash
python run_experiment.py \
    --num-collocation 512 \
    --adam-steps 3000 \
    --lbfgs-iter 2000 \
    --epsilon 0.05 \
    --horizon 5.0
```
Основные аргументы:
- `--num-collocation` — число точек сетки;
- `--adam-steps`, `--lbfgs-iter` — количество шагов оптимизаторов;
- `--epsilon` — параметр релаксации модели;
- `--device` — устройство PyTorch (`cpu`, `cuda:0`, ...);
- `--output` и `--no-show` управляют сохранением и отображением графика.

Скрипт сохранит рисунок в `figures/solow_aif_pinn.png` и покажет сравнительный график AIF-PINN vs. функция Миттага-Леффлера.

## Структура проекта
- `aif_pinn/problem.py` — параметры задачи и правая часть;
- `aif_pinn/data.py` — генерация сетки;
- `aif_pinn/operators.py` — оператор дробной производной;
- `aif_pinn/mittag.py` — аппроксимация функции Миттага-Леффлера;
- `aif_pinn/model.py` — архитектура PINN;
- `aif_pinn/solver.py` — обучение модели;
- `run_experiment.py` — точка входа для воспроизведения эксперимента.

## Как расширять
- Реализуйте собственный `ProblemConfig.rhs`, если нужна правая часть отличная от нуля.
- Создайте новый генератор сетки или модифицируйте `DataGenerator`.
- Экспериментируйте с архитектурой, передав иной набор `hidden_layers` в `AIFPINNModel`.

Удачных экспериментов!
